# Personas

## 1. Eleve
- Objectifs: Consulter planning, notes, absences; telecharger documents.
- Points de douleur: Portails lents, navigation mobile confuse.
- Appareil: Mobile 90% du temps.
- Besoins: Acces rapide, offline, notifications simples.

## 2. Professeur
- Objectifs: Consulter ses cours, suivre classes, deposer documents.
- Points de douleur: Trop de clics pour trouver une classe.
- Appareil: Desktop + mobile.
- Besoins: Vue semaine claire, acces rapide aux classes.

## 3. Scolarite
- Objectifs: Gerer absences, documents officiels, exports.
- Points de douleur: Donnees dispersees, recherches lentes.
- Appareil: Desktop majoritairement.
- Besoins: Tableaux filtrables, actions bulk.

## 4. Responsable pedagogique
- Objectifs: Pilotage pedagogique, validations, suivi notes.
- Points de douleur: Manque de visibilite globale.
- Appareil: Desktop.
- Besoins: KPIs, vues agragees, exports.

## 5. Entreprise / Tuteur
- Objectifs: Suivre etudiant, telecharger documents, consulter infos clefs.
- Points de douleur: Acces complexe, informations cachees.
- Appareil: Mobile.
- Besoins: Acces securise, vue simple, documents.
