# User Journeys

## 1. Consulter son planning (Eleve)
1. Ouvre l'app (offline possible)
2. Authentification ou session active
3. Vue jour (mobile) ou semaine (desktop)
4. Details session (matiere, salle, prof)
5. Changement de jour par swipe

Critere de succes: planning visible en moins de 1.5s et utilisable hors ligne.

## 2. Voir ses notes (Eleve)
1. Accede a l'onglet Notes
2. Liste des evaluations par matiere
3. Affiche la moyenne ponderee
4. Telecharge le bulletin PDF

Critere de succes: donnees lisibles, moyenne correcte, PDF < 500KB.

## 3. Justifier une absence (Eleve -> Scolarite)
1. Eleve ouvre absences
2. Selectionne une absence
3. Upload justificatif (PDF/JPG)
4. Statut passe a "En attente"
5. Scolarite valide ou refuse

Critere de succes: upload reussi, statut clair, historique conserve.
